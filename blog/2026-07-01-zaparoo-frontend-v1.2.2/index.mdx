## What changed

**Frontend fixes:**

- Fixed duplicate controller inputs caused by very fast repeated events.
- Improved game launching responsiveness.
- Preserved hub category focus when backing out from category screens.
- Improved media cover image caching for faster, more stable browsing.
- Added pagination to media directory browsing so large directories load properly.
- Fixed NFC card write modal positioning.
- Fixed updater progress display so stale `100%` progress is not shown. Thanks to [theypsilon](https://github.com/theypsilon).

**Core fixes:**

- Fixed blank browse grids on folder-based systems like PSX.
- Paginated directory entries so large folders of folders no longer stall the app.
- Improved filename tag matching for Arcade games.
- Fixed dynamic cover resizing and WebP conversion.
