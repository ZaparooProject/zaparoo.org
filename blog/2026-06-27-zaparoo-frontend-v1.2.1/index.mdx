[Zaparoo Frontend](/docs/frontend/) v1.2.1 is a hotfix release.

This update fixes an issue where enabling CRT mode would show static snow instead of the Frontend UI. If you updated to v1.2.0 and CRT mode was not usable, update to v1.2.1.

{/* truncate */}

There have also been some issues reported for duplicate media paths showing and cover images being lost after updating. They'll be addressed in another patch update later. If you're experiencing these issues please let us know on Discord!
